Selamat datang di repositori proyek **Siptek-Kantinemama**. Aplikasi ini merupakan solusi digital untuk manajemen pemesanan dan operasional kantin, yang dikembangkan sebagai proyek tugas besar Jurusan Informatika Semester 2 di Universitas Islam Indonesia.

## Tentang Proyek
Siptek-Kantinemama adalah aplikasi berbasis JavaFX yang dirancang untuk mengoptimalkan alur kerja di kantin, mulai dari pemilihan meja, pemesanan menu, hingga pengelolaan transaksi harian secara efisien.

## Persiapan & Instalasi
Aplikasi ini menggunakan Maven sebagai *build tool* dan JavaFX untuk antarmuka pengguna.

### Prasyarat
* **JDK:** Java Development Kit (JDK) 25 atau versi terbaru.
* **Maven:** Pastikan Maven sudah terpasang di sistem Anda.

### Cara Menjalankan Aplikasi
Rekan-rekan dapat menjalankan aplikasi langsung melalui terminal dengan perintah berikut:

1. Run di Launcher.java
2. Membuka File .jar di /target/Siptek-Kantinemama-1.0-SNAPSHOT.jar
